## آدرس پرسیدن و آدرس دادن (Asking directions)


## افعال مدال may و might
- بین فاعل و فعل اصلی قرار می گیرند.
- برای بیان توانایی، اجازه و امکان انجام کاری هستند.
#### ساختار افعال may و might
`شکل ساده فعل + فعل مدال + فاعل`
#### کاربرد افعال may و might
برای بیان احتمال و امکان انجام کاری
```text
He may be outside.
I may play the next match.
They might arrive home now.
We might see you next week. => ما احتمالا هفته آینده شما را ببینیم
```
🎉 فعل مدال `might` امکان وقوع کمتری نسبت ب `may` دارد.
#### روش منفی کردن
با اضافه کردن `not` بعد از فعل مدال انجام می شود.
```text
I may not go to the park.
She might not come to the party.
```
#### روش سوالی کردن
با جابه جایی فعل مدال و فاعل
```text
May they but a house next month ?
Might he become a teacher next year ?
```

## جملات شرطی 1 (Conditional 1)
جملات شرطی از 2 قسمت تشکیل می شوند:
```text
If I feel better, I will go out tonight.
```
- جمله اول شرط است و با `if` شروع می شود.
- جمله دوم جمله اصلی است.
#### شرطی نوع اول
- برای بیان یک شرط ممکن و نتیجه محتمل در آینده
- فعل جمله شرط => حال ساده
- فعل جمله اصلی => آینده ساده

#### ساختار شرطی نوع اول
`فعل + will + فاعل + جمله حال ساده + if`
```text
If it doesn't rain, we will go walking.
If I see him, I'll tell him the news.
If I have time, I'll go shopping.
```
🎉 می توان جای جمله اصلی و جمله شرط را نیز تغییر داد و اگر جمله اصلی در ابتدا باشد، باید ویرگول را حذف کنیم:
```text
I'll go shopping if I have time.
I will go to the park if I finish my work.
```

## مالکیت با of و آپاستروف اس(Of and 'S Possessives)
#### راه های بیان مالکیت
1. استفاده از صفات ملکی
2. استفاده از `of` و `s'`

### مالکیت با s'
برای بیان مالکیت شخص یا حیوان بکار می رود.
#### ساختار مالکیت با s'
`اسم + s' + مالک`
```text
Mina's father
the cat's nose
the children's school
```
🎉 اگر در انتهای اسم `s` جمع باشد:

تنها یک آپاستروف به انتهای آن اضافه می کنیم
```text
students' exam
managers' room
dolphins' brain
```
### مالکیت با of
برای بیان مالکیت اشیا می باشد.
#### ساختار مالکیت با of
`مالک + of + اسم`
```text
the door of the car
the name of the street
the students of the class
the roof of the house
```

## صفات شمارشی 2 (Adjectives of quantity 2)
برای بیان مقدار و تغداد اسم بعد از خود استفاده می شود.

#### تقسیم بندی اسامی از لحاظ شمارش
1. اسامی قابل شمارش

امکان شمردن آن ها وجود دارد و قابل جمع بستن هستند.
```text
one book, two books, three books
```
2. اسامی غیرقابل شمارش

امکان شمردن آن ها وجود ندارد و قابل جمع بستن نیستند.
```text
sugar, bread, water
```
### صفت های شمارشی
1. `a lot of`

به معنای `تعداد خیلی زیاد` یا `مقدار خیلی زیاد` می باشد.

برای اسامی `قابل شمارش جمع` و اسامی `غیرقابل شمارش` استفاده می شود.
```text
a lot of books
a lot of money
```
2. `many`

به معنای `تعداد زیاد` می باشد.

برای اسامی `قابل شمارش جمع` استفاده می شود.
```text
many pens
many apples
```
3. `much`

به معنای `مقدار زیاد` می باشد.

برای اسامی `غیرقابل شمارش` استفاده می شود.
```text
much time
much water
```
4. `a few`

به معنای `تعداد کم` می باشد.

برای اسامی `قابل شمارش جمع` استفاده می شود.
```text
a few chocolate
a few chairs
```
5. `a little`

به معنای `مقدار کم` می باشد.

برای اسامی `غیرقابل شمارش` استفاده می شود.
```text
a little milk
a little cheese
```

## جملات مجهول 1(Passive voice 1)
** جملات معلوم**

فاعل در جمله وجود دارد.

**جملات مجهول**

فاعل در جمله وجود ندارد.

#### ساختار مجهول در زمان حال ساده
`شکل سوم فعل + is / are + مفعول`
```text
He makes butter from milk.
Butter is made from milk.
-------------------------------------------------------------------------
They write the letters. => آن ها نامه ها را می نویسند
The letters are written. => نامه ها نوشته می شوند
```
#### ساختار مجهول در زمان گذشته ساده
`شکل سوم فعل + was / were + مفعول`
```text
She cleaned the house yesterday.
The house was cleaned yesterday.
-------------------------------------------------------------------------
We opened the factory at 9 o'clock. => ما کارخانه را در ساعت 9 باز کردیم
The factory was opened at 9 o'clock. => کارخانه در ساعت 9 باز شد
```

## ساختار های فعل go

1. به همراه حرف اضافه `to`

👈 بعد از آن، مکان مورد نظر قرار می گیرد:
``` text
We will go to England next week.
```
2. به همراه حرف اضافه `on`

🚩 به معنای `ادامه دادن`

👈 بعد از آن، مصدر `ing`دار قرار می گیرد:
``` text
They went on studing until the morning.
```
🚩 به معنای `رفتن`

👈 بعد از آن، اسم قرار می گیرد:
``` text
We will go on a vacation next week.
```
3. بدون هیچ حرف اضافه ای

👈 بعد از آن، مصدر `ing`دار قرار می گیرد:
``` text
I go walking every day.
```

1. بیان تصمیمات از قبل گرفته شده:

``` text
I'm going to fly to Russia on Wednesday.
```

2. بیان قرار ها و برنامه های آینده

✨ زمان حال استمراری

``` text
I'm flying to Russia on Wednesday.
```

## ساختار have و has

به معنای **داشتن** و **خوردن** می باشد.

🎉 فعل `has`:
1. فاعل های سوم شخص مفرد
2. ضمائر فاعلی `he, she, it`

🎉 فعل `have`:
1. فاعل های غیر سوم شخص مفرد
2. ضمائر فاعلی `i, we, you, they`

``` text
He never has dinner late.
```

✨ می توانیم برای بیان مالکیت از `have got` و `has got` نیز استفاده کنیم

``` text
You have got a house.
```

روش منفی کردن `have`و `has`:
1. با قرار دادن `don't` و `doesn't`قبل از آن
2. تبدیل `has` به `have`

``` text
She doesn't have a big house.
```

روش منفی کردن `have got`و `has got`:
1. با قرار دادن `not` بعد از `have` یا `has`

``` text
She hasn't got a book.
```

روش سوالی کردن `have`و `has`:
1. با قرار دادن `do` و `does`در ابتدای جمله
2. تبدیل `has` به `have`

``` text
Does he have an expensive car?
```

روش سوالی کردن `have got`و `has got`:
1. با جابه جایی فاعل با `have / has`

``` text
Have they got many books?
```

## زمان گذشته استمراری

برای بیان عملی:
1. که در گذشته در حال انجام بوده
2. حدود زمانی آن مشخص نیست

🎉 فعل `to be` در زمان حال: `am, is, are`

🎉 فعل `to be` در زمان گذشته: `was, were`

🎉 ساختار زمان گذشته استمراری:

`فاعل + was/were + فعل + ing`

``` text
We were eating breakfast.
```

🎉 فعل `was` برای فاعل های مفرد و `were` برای فاعل های جمع

روش منفی کردن :
1. با اضافه کردن `not` بعد از `was/were`

``` text
They weren't buying clothes.
```

روش سوالی کردن با `wh`دارها :
1. با جابه جایی فاعل و فعل `to be`

``` text
Were they listening to music?
```

روش سوالی کردن با `wh`دارها :
1. قرار دادن کلمه پرسشی در ابتدای جمله
2. قرار دادن فعل `to be`
3. قرار دادن مابقی جمله
``` text
Where were you going?
```

## صفات مقایسه ای

برای مقایسه اشیا، مکان ها و اشخاص

صفات تفضیلی: یکی از طرفین از لحاظ داشتن صفتی، دارای برتری است.

روش ساخت صفت تفضیلی:
1. اگر صفت تک بخشی باشد:

به آخر صفت `er` اضافه میکنیم:
``` text
old => older / young => younger
```
2. اگر صفت چند بخشی باشد:

قبل از صفت `more` اضافه می کنیم:
``` text
expensive => more expensive / beautifull => more beautifull
```
3. اگر حرف آخر صفت `y` باشد: ( بدون در نظر گرفتن تک بخشی یا چند بخشی بودن )

به آخر صفت `er` اضافه میکنیم:
``` text
funny => funnier / pretty => prettier / easy => easier
```
هنگام اضافه کردن `er`، حرف `y` به `i` تبدیل می شود.

روش ساخت جملات مقایسه ای:
1. با اضافه کردن `than` پس از صفت تفضیلی
``` text
This movie is more exciting than that one.
```
2. اگر حرف وسط از جروف صدادار ( `o, i, e, a, u` ) باشد:

حرف آخر آن تکرار می شود:
``` text
thin => thinner / big => bigger
```
صفات تفضیلی بی قاعده:
``` text
good => better / bad => worse / little => less / many/much => more
```
