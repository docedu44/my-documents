## install
برای نصب بهتر است ک خود `create-react-app` را از گلوبال حذف کنیم و بجای آن از `npx` استفاده کنیم تا همیشه آخرین نسخه را برای ما دانلود کند:
```shell
npm un -g create-react-app
# or
yarn global remove create-react-app
```
```shell
npx create-react-app first-project
```
🎉 درون فایل های پروژه های `react` فایلی ب نام `robots.txt` وجود دارد ک مربوط ب خزنده ها می باشد ک می توان مواردی را درون آن ها تعریف کرد.

🎉 همچنین درون `index.js` نیز تابعی ب نام `()reportWebVitals` وجود دارد ک با قرار دادن `()console.log` درون آن، میتوان `performance` برنامه مان را در لاگ ببینیم.

🎉 و فایل `manifest.json` نیز مربوط ب `pwa` می باشد.

### template
برای نصب یک `template` خاص می توان ب صورت زیر از `create-react-app` استفاده کرد:
```shell
npx create-react-app my-project --template [template-name]
```
برای مثال می توان از `typescript` استفاده کرد:

[typescript](/my-documents/docs/Experiences/react/tsx/ts-in-react)

🎉 می توان در `npm` توسط کلمه کلیدی `cra-template` ب دنبال `template` های آماده گشت و از آن ها استفاده کرد.(در واقع `flag` داخل کامند `template--` جایگزین `cra-template` می باشد.)
