## Install
1. برای استفاده از `Vite` می توان ب 2 صورت عمل کرد:
```shell
npm create vite@latest my-app
```
در این حالت از ما می پرسد ک برای چ `framework` یی و با چ زبانی می خواهید پروژه ایجاد کنید.

2. در حالت دوم خودمان تمپلت را مشخص میکنیم:
```shell
npm create vite@latest my-app -- --template vanilla-ts
```
[templates](https://vitejs.dev/guide/#trying-vite-online)

🎉 حال فقط کافیست وارد فولدر پروژه شویم و دستور `npm install` را بزنیم تا وابستگی ها نصب شوند و سپس می توان `npm run dev` را بزنیم تا پروژه اجرا شود.
