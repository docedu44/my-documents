title: GraphQL
---
در واقع مانند `rest` می باشد با این فرق ک می توانیم از فرانت `query` مان را مستقیم ب دیتابیس بزنیم.
همچنین می توان بجای چندین درخواست، با یک درخواست، تمامی اطلاعاتی ک لازم داریم را برگردانیم ک سرعت آن بسیار بالاتر از `rest` می باشد.

🎉 همچنین می توان برای تست و استفاده از `api` های `github` از لینک زیر استفاده کرد، زیر گیتهاب نیز از `graphql` استفاده می کند.

[Github Explorer](https://docs.github.com/en/graphql/overview/explorer)

### Tips
- در حالت کلی ما 2 نوع در خواست داریم: `Queries` ک برای گرفتن داده استفاده می شود و `Mutations` ک برای تغییر داده استفاده می شود.
