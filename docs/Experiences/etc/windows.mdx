
### kill port
🎉 برای بستن یک `port` خاص در ویندوز می توان از 2 روش زیر استفاده کرد:

1. استفاده از خود دستورات ویندوز
```shell
netstat -ano | findstr :<port> # آن pid پیدا کردن پورت مورد نظر و
taskkill /PID <port pid> /F # را بزنیم تا پورت بسته شود pid در واقع در دستور قبلی باید عدد داده شده ب عنوان
```
🎉 حتما باید در حالت `adminstrator` محیط `cmd` را باز کنیم(در محیط `run` باید `cmd` را وارد کرده سپس دکمه های `ctrl + shift + enter` را می زنیم)

2. استفاده از پکیج های `node`
```shell
npx kill-port <port>
```
