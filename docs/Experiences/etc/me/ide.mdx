## Webstorm
- برای `zoom in` و `zoom out` می توان از دکمه های زیر استفاده کرد:
``` text
Shift + Alt + > // Zoom In
Shift + Alt + < // Zoom Out
```
- برای نمایش آخرین تغییر یا آخرین محلی ک روی آن کار می کردیم، می توان از دکمه های زیر استفاده کرد:
``` text
Shift + Ctrl + E
```
